Expressions are essentially questions you pose to [[JavaScript]] , and they are designed to return a value as an answer [[expressions always results in a single value]].  Expressions can involve operations, calculations, or function calls.
examples:
  ```
  7; // literal expression, because we are really writing the value
  console.log(7+3);
  let x = 5*2
  randomFunction(); //the function must be defined first
  ```
