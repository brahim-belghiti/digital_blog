# Abex

Abex is a modern landing page designed to connect users with shipping agencies and carriers. As the frontend developer, I built a clean, responsive, and user-friendly interface that effectively showcases the agency's services while ensuring accessibility and performance.

## üåü Project Overview

Abex serves as a digital storefront for a transportation agency, providing users with clear information about shipping services and an easy way to get in touch. Built with Next.js, the landing page leverages modern web technologies to deliver a fast, accessible, and globally adaptable experience.

## ‚ú® Key Features

- **Multilingual Support**: Powered by i18next for seamless internationalization.
- **Contact Form**: User-friendly form with performant validation using React Hook Form.
- **Email Integration**: Direct email functionality via Nodemailer, eliminating the need for a backend API.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Fast Performance**: Leverages Next.js for quick load times and smooth navigation.
